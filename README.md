### 项目介绍

qq_music是基于原生js开发的项目，他是一个测试项目。

### 使用方法

- 在集成终端中打开QQMusicApi→npm start
- 打开网页

### 功能描述

#### 已经实现的功能：

- 推荐 

- - 首页 

- - - 歌单推荐(达人歌单(可拖拽)，专区，官方歌单)
    - 新歌首发(可拖拽)
    - 精彩推荐(可拖拽)
    - 排行榜
    - 底部栏

- - 搜索 

- - - 搜索历史
    - 热门搜索
    - 搜索展示（分为两种搜索，一种按下回车键，一种点击搜索按钮）

### 前端技术

- html+less+js
